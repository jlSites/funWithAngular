Play with CSS!

CSS stands for Cascading Style Sheets.

CSS describes how HTML elements are to be displayed on screen, paper, or in other media.

For CSS syntax, a CSS rule-set consists of a selector and a declaration block.

CSS selectors are used to find HTML elements based on their element name, id, class, attribute, and more.

  • The element selector (elementname {})
  • The id selector (#idstring {})
  • The class selector (.classname {})

Three ways to insert CSS.

  • External style sheet (link within head element)
  • Internal style sheet (style element within head element)
  • Inline style (style attribute to the relevant element)

Cascading Order.

  1. Inline style (inside an HTML element)
  2. External and internal style sheets (in the head section)
  3. Browser default
